【安裝破解教學】

先將防毒軟體關閉再安裝！

1. 解壓縮後，掛載光碟映像檔，執行 OInstall.exe

2. [首頁]
   →選擇[Microsoft Office 2019]
   →選擇[x86 或 x64]
   →選擇 語言 [en-US] 或 [zh-CN] 或 [zh-TW]

   →安裝全套軟體 → 勾選[ProPlus:]
   →只想安裝某幾項軟體 → 勾選[單個產品] → 勾選 [Word]、[PowerPoint]、[ProjectPro] ...等等

   →[更新通道]選擇: [Monthly Channel]
   →點擊[安裝 Office]

3. 安裝完畢，需聯網 → 再破解
   選擇[實用程序]
   →勾選[在線 KMS]
   
   →點擊[[Office 零售轉批量]
   →畫面出現 [===Office轉換中...] → 等待 → 出現 [******已完成******]
   
   →再點擊 [激活 Office] → 等待 → 出現 [<Product activation successful>]
   →完成，結束
